# Cyber Infra Dashboard

This is the React frontend for the SaaS security scanner.

## Setup

1. Install dependencies:
   ```sh
   cd dashboard
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Structure
- `src/`: React app source code
- `App.js`: Main dashboard UI

The dashboard will connect to the API and display scan results.
