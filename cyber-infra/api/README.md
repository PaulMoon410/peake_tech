# Cyber Infra API

This is the backend for the SaaS security scanner. It uses Node.js (Express) and PostgreSQL.

## Setup

1. Install dependencies:
   ```sh
   cd api
   npm install
   ```
2. Set up your PostgreSQL database and update `.env` with your credentials.
3. Start the server:
   ```sh
   npm run dev
   ```

## Structure
- `index.js`: Main API entry point
- `.env`: Environment variables
- `package.json`: Project config

Scanner modules and routes will be added soon.
